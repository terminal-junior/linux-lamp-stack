# linux-lamp-stack
Guia prático e didático para instalação e configuração do stack LAMP (Linux, Apache, MySQL/MariaDB e PHP) em diferentes distribuições Linux. O projeto tem foco em aprendizado, boas práticas de infraestrutura, organização e automação, sendo ideal para estudantes, iniciantes em SysAdmin/DevOps e profissionais que desejam padronizar ambientes LAMP.
